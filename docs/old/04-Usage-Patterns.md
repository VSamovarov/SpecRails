# SpecRails — Usage Patterns

## Основной сценарий
1. Пользователь пишет текст
2. Выделяет фрагмент или ставит курсор
3. `/form` (пример)
4. Уточняет цель
5. Просмотр результата
6. Принять / редактировать / регенерировать

## Форматы вывода
- JSON
- YAML
- Markdown
- Таблицы и диаграммы

## Где используется
- VS Code
- Chat (slash‑команды)
- CLI (опционально)
